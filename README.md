Description

TABLES is a web project that demonstrates dynamic table creation and manipulation using HTML, CSS, and JavaScript.
The application allows users to generate, update, and interact with table content based on JavaScript logic.
It showcases how to work with the Document Object Model (DOM), event handling, and data rendering inside HTML tables.

Features
Dynamic creation and manipulation of HTML tables using JavaScript
Interactive and responsive user interface
Clear separation of structure, style, and behavior with HTML, CSS, and JavaScript
Practical example of DOM manipulation and data presentation

Who this is for
This project is designed for beginners and intermediate developers who want to learn or practice dynamic HTML table handling and core JavaScript concepts. 
It is also a useful reference for understanding how to update the user interface using JavaScript.
